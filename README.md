|  [뒤로가기]                                  |
---------------------------------------------
🧩 파일 구조와 역할
src/
 ├─ gui/                ← 진호 씨 담당
 │     ├─ main_window.py   (메인 UI)
 │     ├─ charts.py         (그래프)
 │     └─ components.py     (버튼, 카드 등 - 선택)
 │
 ├─ ai/                 ← 나지마 담당
 │     ├─ predictor.py
 │     ├─ model_manager.py
 │     └─ preprocessing.py
 │
 └─ app.py              ← GUI ←→ AI 연결
🤝 두 기능이 연결되는 방식
진호씨(main_window.py) → 버튼 클릭
       ↓
   app.py → 나지마(AI) predictor.predict()
       ↓
최종 예측 결과 UI에 출력
